2. Do not use public storage for personal data

You can accidentally provide additional information not only through social networks. For example, there is no need to store sensitive data in online services designed for information exchange. For example, Google Docs is not the best place for a password file, and scanned passports do not need to be uploaded to Dropbox (unless they are pre-packaged in an encrypted archive).
